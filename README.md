# FinTrack — Expense Tracker & Analytics Dashboard

A professional-grade personal finance tracker built with pure HTML, CSS, and JavaScript. No frameworks, no backend — runs entirely in the browser.

## Features

- **Dashboard** — Income vs Expense summary cards, monthly bar chart, category donut chart, recent transactions
- **Transactions** — Full transaction table with filter by type/category and live search
- **Analytics** — Category bar chart, income/expense trend line chart, top spending categories, monthly summary
- **Budget Planner** — Set total and per-category budgets with visual progress bars and 90% alert
- **Export CSV** — Download all transactions as a spreadsheet
- **LocalStorage** — Data persists between sessions, no login needed
- **Keyboard Shortcut** — Press `N` to quickly add a transaction

## Tech Stack

- HTML5, CSS3 (CSS Variables, Grid, Flexbox, Animations)
- Vanilla JavaScript (ES6+)
- Chart.js v4 (CDN)
- Google Fonts (Syne + DM Mono)
- LocalStorage API

## How to Run

Just open `index.html` in any browser — no installation needed.

```
double-click index.html
```
